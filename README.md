# sumorobot-manager

Update SumoFirmware

## How to run

1. Install Python3, PyQt5, pyserial (terminal: pip3 install pyqt5 pyserial)
2. Launch the app (terminal: python3 main.py)

## How to use

1. Plug in SumoRobot via a micro USB cable
2. Press Update Firmware
